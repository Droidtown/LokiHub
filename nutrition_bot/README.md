# Nutrient Calculator Chatbot 營養計算小幫手
***
## 簡介
Nutrient Calculator是一款可以根據年齡及性別推薦每日需要營養素(如：鈣、維生素C)的聊天機器人。
***
## 操作指南
請簡單的和機器人打招呼以開始對話，並在每一次說話前先`@Bot`。<br>
[實際操作影片](https://youtu.be/TabdABk-N-Q)
